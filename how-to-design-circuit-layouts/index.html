How to Design Circuit Layouts

Designing a Circuit Diagram is easy but designing a Circuit Layout is difficult Task.Generally students like us prefer manual method for designing layout in which circuit layout is drawn on paper and then mirror image of the same is drawn on the PCB using permanent marker.Design circuit layouts using software is difficult but we can’t escape our self when we are talking about a big project consisting of numerous components.In such cases preparing layout using manual method is tedious venture.

For our final year degree project we tried to make layout using traditional manual method but,we found it difficult to make a layout for forty pin IC on the PCB.Then we decided to go for software method.No one among us was familiar about the software method.Initially we faced lots of difficulty, then finally we came out with shortcut method of designing layout using “EAGLE SOFTWARE” along with some additional softwares.

For Designing a Circuit Layout We are Using Eagle as a Layout Maker.

Eagle:

Eagle offers user friendly, powerful and affordable solutions for PCB design, including Schematic Capture, Board Layout and Autorouter. User Language Programs enable features such as simulation and 3D-visualization.

Note:Making Layout for project free version of Eagle is sufficient.

Screen Capture Tools:

We are trying to give you a shortest way for making layout.For this purpose use FastStone as Image Capture Software,You can use any image capturing software for this purpose.

Download FastStone From Here: Download FastStone.

Note: FastStone Capture is not a free program so instead of using FastStone Capture.

Windows7 User use a Snipping Tool which is present in Start→All Programs→Accessories→Snipping Tool

Use Greenshot which is a Free Software which is equivalent to FastStone.

Download Greenshot From Here: Download Greenshot.
