Setup Octopress On Windows7

Assalamualaikum,

if you love static pages or not have enough money to buy a host to run a powerfull wordpress blog then Octopress is a best software for start blogging.Its very simple and clean but due to unfamiliar to run a running shell commands and basics of Git,its very difficult to setup a Octopress blog specially on a Windows7 machines.

This is quick guide to setting up Octopress on a Windows7 machine to publish to GitHub pages.

Before You Begin

1.Install Git.

2.Install Ruby Installer For windows.In this tutorial i use Ruby 1.9.3-p194

3.Install Development Kit.In this tutorial i use DevKit-tdm-32-4.5.2-20111229-1559-sfx.exe and extract files into C:/RubyDevKit.

4.create a folder in C drive like github
